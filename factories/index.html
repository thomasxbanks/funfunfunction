JS Factory Functions

As you can see, not only is the factory function shorter than the class, if you clickety-clickety the buttons, you will see that foxes are more talkative than dogs!

Factory

const fox = () => {
  const sound = 'ring ding ding ding'
  return {
    talk: () => $('#message_1').append(sound)
  }
}
const megan = fox() // megan is a new fox
megan.talk() // run on load
$('#button_1').click(megan.talk) // run on click

Class

class Barker {
  constructor() {
    this.sound = 'bark'
  }
  talk() {
    $('#message_2').append(this.sound)
  }
}
const sue = new Barker() // sue is a new dog
sue.talk() // run on load
$('#button_2').click(sue.talk) // run on click
